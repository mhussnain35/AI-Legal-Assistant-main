# AI Legal Assistant

An AI-powered Legal Assistant designed to provide legal information, answer legal queries, and assist users with legal document understanding through an intelligent conversational interface.

## Overview

AI Legal Assistant is a web-based application that leverages Artificial Intelligence to help users access legal information efficiently. The system provides an interactive chatbot interface capable of answering legal questions, guiding users through legal processes, and improving access to legal knowledge.

## Features

* AI-powered legal chatbot
* Natural language query processing
* User-friendly interface
* Real-time legal assistance
* Secure and responsive design

## Installation

### Clone the Repository

```bash
git clone https://github.com/mhussnain35/AI-Legal-Assistant-main.git
cd AI-Legal-Assistant-main
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / macOS

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```
